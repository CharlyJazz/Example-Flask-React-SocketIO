# File Input make in React with SocketIO and Flask

### About
___
Tool for uploading multiple Zips and Files Asynchronously. Make on React.JS with Flask and SocketIO.


###  Installing and Running:
____
  ### 1. Server 
``` bash
$ pipenv install

$ pipenv shell

$ export FLASK_APP=server.py

$ flask run

```
### 2. The App
``` bash
$ yarn or npm install

$ yarn or npm start
```
